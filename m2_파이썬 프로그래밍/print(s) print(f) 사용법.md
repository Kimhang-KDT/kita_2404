# print(list1[0:3])
* 리스트의 인덱스 0~3까지를 출력하라는 의미

list1 = [58, 45, 69, 19, 4, 87, 29, 13, 39, 15, 54, [8, 33, 11], 27, 49, 63, 98, 22, 82]

# list1[0::2] list1[6::-1] list1[1:9:2]
# [시작점:끝점:스텝(-는 역순)]

* 2부터 끝까지인데 -1은 꺼꾸로라는 뜻\
* 그래서 :: 두번쓴다. :한번쓰면 끝까지라는 뜻인데 :를 하나 더 붙여서 -1꺼꾸로라는 의미를 부여했다.\
* print('69,45,58출력 : ', list1[2::-1])\
* 0부터 끝까지인데 2스텝으로\
* print(list1[0::2], "\n")\

* 3번째부터 거꾸로 끝까지 출력\
print(list1[2::-1])\
* 3번째부터 한칸씩 띄고 끝까지 출력\
print(list1[2::2])\
* 6번째부터 두칸씩 띄고 끝까지 출력\
print(list1[5::3])\
* 처음부터 끝까지인데 역순\
print(list1[::-1])
* 시작점과 끝점, 역순이면\
print(list1[9:4:-1])
